еӯҳеӮЁеҷЁзі»з»ҹпјҡдё»еӯҳгҖҒCacheгҖҒиҷҡжӢҹеӯҳеӮЁеҷЁдёҺеӨ–йғЁеӯҳеӮЁеұӮж¬ЎгҖӮ

3.1 еӯҳеӮЁеҷЁжҰӮиҝ°

3.1.1 еӯҳеӮЁеҷЁзҡ„еҲҶзұ»

1. жҢүеӯҳеӮЁе…ғд»¶еҲҶзұ»

  • еҚҠеҜјдҪ“еӯҳеӮЁеҷЁпјҡз”ЁеҚҠеҜјдҪ“еҷЁд»¶жһ„жҲҗгҖӮ
  • зЈҒиЎЁйқўеӯҳеӮЁеҷЁпјҡеҰӮзЈҒзӣҳгҖҒзЈҒеёҰгҖӮ
  • е…үд»ӢиҙЁеӯҳеӮЁеҷЁпјҡе…үзӣҳеӯҳеӮЁеҷЁгҖӮ

2. жҢүеӯҳеҸ–ж–№ејҸеҲҶзұ»

зұ»еһӢ зү№зӮ№ зӨәдҫӢ
йҡҸжңәеӯҳеҸ–еӯҳеӮЁеҷЁ(RAM) жҢүең°еқҖи®ҝй—®пјҢи®ҝй—®ж—¶й—ҙеӣәе®ҡдё”зӣёзӯү CacheгҖҒдё»еӯҳ
йЎәеәҸеӯҳеӮЁеӯҳеӮЁеҷЁ(SAM) дҝЎжҒҜжҢүйЎәеәҸеӯҳж”ҫе’ҢиҜ»еҮәпјҢеӯҳеҸ–ж—¶й—ҙеҸ–еҶідәҺдҪҚзҪ® зЈҒеёҰеӯҳеӮЁеҷЁ
зӣҙжҺҘеӯҳеҸ–еӯҳеӮЁеҷЁ(DAM) е…ҲйҡҸжңәйҖүеҸ–еҢәеҹҹпјҢеҶҚйЎәеәҸи®ҝй—® зЈҒзӣҳеӯҳеӮЁеҷЁ
зӣёиҒ”еӯҳеӮЁеҷЁ(AM) жҢүеҶ…е®№жЈҖзҙўпјҢд№ҹз§°жҢүеҶ…е®№и®ҝй—®еӯҳеӮЁеҷЁ(CAM) еҝ«иЎЁ(TLB)

3. жҢүдҝЎжҒҜзҡ„еҸҜжӣҙж”№жҖ§еҲҶзұ»

  • иҜ»еҶҷеӯҳеӮЁеҷЁпјҡж—ўиғҪиҜ»еҸҲиғҪеҶҷгҖӮ
  • **еҸӘиҜ»еӯҳеӮЁеҷЁ(ROM)**пјҡйҖҡеёёеҸӘиҜ»дёҚеҶҷпјҢжҹҗдәӣжғ…еҶөдёӢеҸҜйҮҚеҶҷгҖӮ

4. жҢүж–ӯз”өеҗҺдҝЎжҒҜзҡ„еҸҜдҝқеӯҳжҖ§еҲҶзұ»

  • йқһжҳ“еӨұжҖ§еӯҳеӮЁеҷЁпјҡж–ӯз”өеҗҺдҝЎжҒҜдҝқжҢҒпјҲROMгҖҒзЈҒиЎЁйқўгҖҒе…үеӯҳеӮЁеҷЁпјүгҖӮ
  • жҳ“еӨұжҖ§еӯҳеӮЁеҷЁпјҡж–ӯз”өеҗҺдҝЎжҒҜдёўеӨұпјҲRAMгҖҒCacheпјүгҖӮ

5. жҢүеҠҹиғҪеҲҶзұ»

зұ»еһӢ зү№зӮ№ йҖҹеәҰ е®№йҮҸ
й«ҳйҖҹзј“еҶІеӯҳеӮЁеҷЁ(Cache) дҪҚдәҺдё»еӯҳе’ҢCPUд№Ӣй—ҙпјҢеӯҳж”ҫеҪ“еүҚеёёз”ЁжҢҮд»Өе’Ңж•°жҚ® жңҖеҝ« жңҖе°Ҹ
дё»еӯҳеӮЁеҷЁ(дё»еӯҳ) CPUеҸҜзӣҙжҺҘйҡҸжңәи®ҝй—®пјҢеӯҳж”ҫеҪ“еүҚиҝҗиЎҢзҡ„зЁӢеәҸе’Ңж•°жҚ® иҫғеҝ« иҫғе°Ҹ
иҫ…еҠ©еӯҳеӮЁеҷЁ(иҫ…еӯҳ) еӯҳж”ҫжҡӮж—¶дёҚз”Ёзҡ„зЁӢеәҸе’Ңж•°жҚ®пјҢйңҖи°ғе…Ҙдё»еӯҳеҗҺжүҚиғҪи®ҝй—® жңҖж…ў жңҖеӨ§

3.1.2 еӯҳеӮЁеҷЁзҡ„жҖ§иғҪжҢҮж Ү

жҢҮж Ү е®ҡд№ү
еӯҳеӮЁе®№йҮҸ еӯҳеӮЁеӯ—ж•° Г— еӯҳеӮЁеӯ—й•ҝ
еҚ•дҪҚжҲҗжң¬ жҖ»жҲҗжң¬ / жҖ»е®№йҮҸ
еӯҳеҸ–ж—¶й—ҙ($T_a$) еҗҜеҠЁдёҖж¬ЎеӯҳеӮЁеҷЁж“ҚдҪңеҲ°е®ҢжҲҗжүҖйңҖж—¶й—ҙ
еӯҳеҸ–е‘Ёжңҹ($T_m$) еӯҳеӮЁеҷЁиҝһз»ӯиҜ»еҶҷж“ҚдҪңе…Ғи®ёзҡ„жңҖзҹӯй—ҙйҡ”ж—¶й—ҙпјҢ$T_m = T_a +$ йҷ„еҠ ж—¶й—ҙ
дё»еӯҳеёҰе®Ҫ(B) ж•°жҚ®дј иҫ“йҖҹзҺҮпјҢеҚ•дҪҚпјҡеӯ—/з§’гҖҒB/sгҖҒb/s

3.1.3 еұӮж¬ЎеҢ–еӯҳеӮЁеҷЁзҡ„еҹәжң¬з»“жһ„

  • зӣ®зҡ„пјҡе…јйЎҫеӨ§е®№йҮҸгҖҒй«ҳйҖҹеәҰгҖҒдҪҺжҲҗжң¬гҖӮ
  • еұӮж¬Ўз»“жһ„пјҡ
    • Cache-дё»еӯҳеұӮпјҡи§ЈеҶійҖҹеәҰдёҚеҢ№й…Қй—®йўҳпјҢз”ұзЎ¬д»¶иҮӘеҠЁе®ҢжҲҗпјҢеҜ№жүҖжңүзЁӢеәҸе‘ҳйҖҸжҳҺгҖӮ
    • дё»еӯҳ-иҫ…еӯҳеұӮпјҡи§ЈеҶіе®№йҮҸй—®йўҳпјҢз”ұзЎ¬д»¶е’ҢOSе…ұеҗҢе®ҢжҲҗпјҢеҜ№еә”з”ЁзЁӢеәҸе‘ҳйҖҸжҳҺгҖӮ

3.2 дё»еӯҳеӮЁеҷЁ

3.2.1 SRAMе’ҢDRAM

зұ»еһӢ еӯҳеӮЁе…ғ зү№зӮ№ з”ЁйҖ”
SRAM еҸҢзЁіжҖҒи§ҰеҸ‘еҷЁ йқһз ҙеқҸжҖ§иҜ»еҮәпјҢдёҚйңҖеҲ·ж–°пјҢйҖҹеәҰеҝ«пјҢйӣҶжҲҗеәҰдҪҺпјҢд»·ж јй«ҳ Cache
DRAM жҷ¶дҪ“з®Ў з ҙеқҸжҖ§иҜ»еҮәпјҢйңҖе®ҡж—¶еҲ·ж–°пјҢйҖҹеәҰж…ўпјҢйӣҶжҲҗеәҰй«ҳпјҢд»·ж јдҪҺ дё»еӯҳ

DRAMзҡ„еҲ·ж–°

  • еҲ·ж–°еҚ•дҪҚпјҡиЎҢгҖӮ
  • еҲ·ж–°е‘ЁжңҹпјҡйҖҡеёё2msгҖӮ
  • еҲ·ж–°ж–№ејҸпјҡ
    • йӣҶдёӯеҲ·ж–°пјҡеӣәе®ҡж—¶й—ҙеҲ·ж–°жүҖжңүиЎҢпјҢеӯҳеңЁвҖңжӯ»еҢәвҖқпјҲи®ҝеӯҳжӯ»еҢәпјүгҖӮ
    • еҲҶж•ЈеҲ·ж–°пјҡжҜҸдёӘе·ҘдҪңе‘ЁжңҹеҲҶдёӨйғЁеҲҶпјҲиҜ»еҶҷ+еҲ·ж–°пјүпјҢж— жӯ»еҢәпјҢдҪҶеӯҳеҸ–е‘Ёжңҹе»¶й•ҝгҖӮ
    • ејӮжӯҘеҲ·ж–°пјҡз»“еҗҲдёӨиҖ…пјҢеҲ·ж–°е‘ЁжңҹеҶ…еқҮеҢҖеҲ·ж–°еҗ„иЎҢгҖӮ

DRAMең°еқҖеј•и„ҡеӨҚз”Ё

  • иЎҢең°еқҖе’ҢеҲ—ең°еқҖеҲҶдёӨж¬ЎйҖҡиҝҮзӣёеҗҢеј•и„ҡиҫ“е…ҘпјҢеҮҸе°‘еј•и„ҡж•°гҖӮ

3.2.2 еҸӘиҜ»еӯҳеӮЁеҷЁ(ROM)

зұ»еһӢ зү№зӮ№
MROM жҺ©жЁЎејҸпјҢеҶ…е®№еңЁиҠҜзүҮз”ҹдә§ж—¶еҶҷе…ҘпјҢдёҚеҸҜж”№
PROM дёҖж¬ЎеҸҜзј–зЁӢпјҢз”ЁжҲ·еҶҷе…ҘдёҖж¬ЎпјҢдёҚеҸҜж”№
EPROM еҸҜж“ҰйҷӨеҸҜзј–зЁӢпјҢеӨҡж¬Ўж”№еҶҷпјҢдҪҶеҶҷе…Ҙж¬Ўж•°жңүйҷҗ
Flash еҸҜй•ҝжңҹдҝқеӯҳпјҢеҝ«йҖҹж“ҰйҷӨе’ҢйҮҚеҶҷ
SSD еӣәжҖҒзЎ¬зӣҳпјҢз”ұжҺ§еҲ¶еҚ•е…ғе’ҢFlashз»„жҲҗ

3.2.3 дё»еӯҳеӮЁеҷЁзҡ„еҹәжң¬з»„жҲҗ

  • еӯҳеӮЁдҪ“пјҡеӯҳеӮЁеҚ•е…ғзҡ„йӣҶеҗҲгҖӮ
  • ең°еқҖиҜ‘з ҒпјҡдёҖз»ҙиҜ‘з ҒпјҲеӯ—зәҝеӨҡпјүгҖҒдәҢз»ҙиҜ‘з ҒпјҲиЎҢ+еҲ—иҜ‘з ҒпјүгҖӮ
  • зүҮйҖүжҺ§еҲ¶дҝЎеҸ·пјҡйҖүдёӯиҰҒи®ҝй—®зҡ„иҠҜзүҮгҖӮ
  • иҜ»еҶҷжҺ§еҲ¶дҝЎеҸ·пјҡжҺ§еҲ¶иҜ»/еҶҷж“ҚдҪңгҖӮ

CPUи®ҝеӯҳиҝҮзЁӢпјҡ

  1. CPUе°Ҷең°еқҖйҖҒMAR вҶ’ ең°еқҖзәҝ вҶ’ дё»еӯҳең°еқҖеҜ„еӯҳеҷЁ вҶ’ ең°еқҖиҜ‘з ҒгҖӮ
  2. CPUе°ҶиҜ»еҶҷдҝЎеҸ·йҖҒжҺ§еҲ¶зәҝ вҶ’ дё»еӯҳиҜ»еҶҷжҺ§еҲ¶з”өи·ҜгҖӮ
  3. иҜ»ж“ҚдҪңпјҡдё»еӯҳиҜ»еҮәж•°жҚ® вҶ’ ж•°жҚ®зәҝ вҶ’ MDRгҖӮ
  4. еҶҷж“ҚдҪңпјҡCPUе°Ҷж•°жҚ®йҖҒMDR вҶ’ ж•°жҚ®зәҝ вҶ’ дё»еӯҳеҶҷе…ҘйҖүдёӯеҚ•е…ғгҖӮ

3.2.4 еӨҡжЁЎеқ—еӯҳеӮЁеҷЁ

1. иҝһз»ӯзј–еқҖж–№ејҸ

  • й«ҳдҪҚдёәжЁЎеқ—еҸ·пјҢдҪҺдҪҚдёәжЁЎеқ—еҶ…ең°еқҖгҖӮ
  • жЁЎеқ—еҶ…ең°еқҖиҝһз»ӯпјҢжЁЎеқ—й—ҙдёІиЎҢи®ҝй—®пјҢдёҚиғҪжҸҗй«ҳеҗһеҗҗзҺҮгҖӮ

2. дәӨеҸүзј–еқҖж–№ејҸ

  • дҪҺдҪҚдёәжЁЎеқ—еҸ·пјҢй«ҳдҪҚдёәжЁЎеқ—еҶ…ең°еқҖгҖӮ
  • жЁЎеқ—й—ҙдәӨеҸүзј–еқҖпјҢеҸҜе№¶иЎҢи®ҝй—®гҖӮ
  • иҪ®жөҒеҗҜеҠЁпјҡжҜҸйҡ” $1/m$ дёӘеӯҳеӮЁе‘ЁжңҹеҗҜеҠЁдёҖдёӘжЁЎеқ—пјҢеҗһеҗҗзҺҮжҸҗй«ҳ $m$ еҖҚгҖӮ
  • еҗҢж—¶еҗҜеҠЁпјҡжүҖжңүжЁЎеқ—еҗҢж—¶еҗҜеҠЁпјҢдёҖж¬ЎжҸҗдҫӣ $m \times$ еӯ—й•ҝж•°жҚ®гҖӮ

3.3 дё»еӯҳеӮЁеҷЁе’ҢCPUзҡ„иҝһжҺҘ

3.3.1 дҪҚжү©еұ•

  • зӣ®зҡ„пјҡеўһеҠ еӯҳеӮЁеӯ—й•ҝпјҲдҪҚж•°пјүгҖӮ
  • иҝһжҺҘж–№ејҸпјҡең°еқҖзәҝгҖҒиҜ»еҶҷжҺ§еҲ¶зәҝгҖҒзүҮйҖүзәҝеҜ№еә”зӣёиҝһпјҢж•°жҚ®зәҝеҚ•зӢ¬еј•еҮәгҖӮ

3.3.2 еӯ—жү©еұ•

  • зӣ®зҡ„пјҡеўһеҠ еӯҳеӮЁе®№йҮҸпјҲеӯ—ж•°пјүгҖӮ
  • иҝһжҺҘж–№ејҸпјҡең°еқҖзәҝдҪҺдҪҚзӣёиҝһпјҢзүҮйҖүдҝЎеҸ·з”ұй«ҳдҪҚең°еқҖз»ҸиҜ‘з ҒеҷЁдә§з”ҹгҖӮ

3.3.3 еӯ—дҪҚеҗҢж—¶жү©еұ•

  • зӣ®зҡ„пјҡж—ўжү©еұ•е®№йҮҸеҸҲжү©еұ•дҪҚж•°гҖӮ
  • иҝһжҺҘж–№ејҸпјҡдҪҚжү©еұ•зҡ„иҠҜзүҮдҪңдёәдёҖз»„пјҢз»„еҶ…дҪҚжү©еұ•пјҢз»„й—ҙеӯ—жү©еұ•гҖӮ

3.4 еӨ–йғЁеӯҳеӮЁеҷЁ

3.4.1 зЈҒзӣҳеӯҳеӮЁеҷЁ

з»„жҲҗ

  • зЈҒзӣҳй©ұеҠЁеҷЁпјҡй©ұеҠЁзЈҒзӣҳиҪ¬еҠЁпјҢиҝӣиЎҢиҜ»еҶҷгҖӮ
  • зЈҒзӣҳжҺ§еҲ¶еҷЁпјҡдё»жңәдёҺзЈҒзӣҳй©ұеҠЁеҷЁзҡ„жҺҘеҸЈгҖӮ
  • зӣҳзүҮпјҡеӯҳеӮЁд»ӢиҙЁгҖӮ

еӯҳеӮЁеҢәеҹҹ

  • и®°еҪ•йқўпјҲзЈҒеӨҙж•°пјүпјҡзӣҳзүҮж•°йҮҸ Г— 2гҖӮ
  • зЈҒйҒ“пјҡи®°еҪ•йқўдёҠзҡ„еҗҢеҝғеңҶгҖӮ
  • жҹұйқўпјҡдёҚеҗҢи®°еҪ•йқўзӣёеҗҢдҪҚзҪ®зҡ„зЈҒйҒ“йӣҶеҗҲгҖӮ
  • жүҮеҢәпјҲеқ—пјүпјҡзЈҒзӣҳиҜ»еҶҷзҡ„жңҖе°ҸеҚ•дҪҚгҖӮ

жҖ§иғҪжҢҮж Ү

жҢҮж Ү е®ҡд№ү
и®°еҪ•еҜҶеәҰ йҒ“еҜҶеәҰпјҲеҫ„еҗ‘еҚ•дҪҚй•ҝеәҰзЈҒйҒ“ж•°пјүгҖҒдҪҚеҜҶеәҰпјҲзЈҒйҒ“еҚ•дҪҚй•ҝеәҰдҪҚж•°пјү
зЈҒзӣҳе®№йҮҸ ж јејҸеҢ–е®№йҮҸ = и®°еҪ•йқўж•° Г— жҹұйқўж•° Г— жҜҸйҒ“жүҮеҢәж•° Г— жүҮеҢәе®№йҮҸ
еӯҳеҸ–ж—¶й—ҙ еҜ»йҒ“ж—¶й—ҙ + ж—ӢиҪ¬е»¶иҝҹж—¶й—ҙ + дј иҫ“ж—¶й—ҙпјҲйҖҡеёёеҸ–е№іеқҮеҖјпјү
ж•°жҚ®дј иҫ“йҖҹзҺҮ еҚ•дҪҚж—¶й—ҙдј йҖҒзҡ„ж•°жҚ®йҮҸ

RAID5 еҘҮеҒ¶ж ЎйӘҢпјҢеҲҶеёғејҸж ЎйӘҢеқ—пјҢе…јйЎҫжҖ§иғҪдёҺе®№й”ҷ

3.4.2 еӣәжҖҒзЎ¬зӣҳ(SSD)

  • з»„жҲҗпјҡй—ӘеӯҳиҠҜзүҮ + й—Әеӯҳзҝ»иҜ‘еұӮпјҲжҺ§еҲ¶еҷЁпјүгҖӮ
  • з»“жһ„пјҡеқ—пјҲBlockпјүвҶ’ йЎөпјҲPageпјүпјҢиҜ»еҶҷд»ҘйЎөдёәеҚ•дҪҚгҖӮ
  • еҶҷж“ҚдҪңпјҡиӢҘйЎөйқһз©әпјҢйңҖе°ҶжүҖеңЁеқ—жңүж•ҲйЎөеӨҚеҲ¶еҲ°ж–°еқ—пјҢеҶҚеҶҷе…ҘпјҢеҶҚж“ҰйҷӨеҺҹеқ—гҖӮ
  • зЈЁжҚҹеқҮиЎЎпјҡеҠЁжҖҒзЈЁжҚҹеқҮиЎЎпјҲеҶҷж—¶йҖүиҫғж–°еқ—пјүгҖҒйқҷжҖҒзЈЁжҚҹеқҮиЎЎпјҲеҲҶй…ҚиҖҒеқ—еӯҳйқҷжҖҒж•°жҚ®пјүгҖӮ

3.5 й«ҳйҖҹзј“еҶІеӯҳеӮЁеҷЁ(Cache)

3.5.1 зЁӢеәҸи®ҝй—®зҡ„еұҖйғЁжҖ§еҺҹзҗҶ

  • ж—¶й—ҙеұҖйғЁжҖ§пјҡеҪ“еүҚдҪҝз”Ёзҡ„дҝЎжҒҜиҝ‘жңҹеҸҜиғҪеҶҚж¬ЎдҪҝз”ЁгҖӮ
  • з©әй—ҙеұҖйғЁжҖ§пјҡиҝ‘жңҹиҰҒз”Ёзҡ„дҝЎжҒҜдёҺеҪ“еүҚдҝЎжҒҜеңЁз©әй—ҙдёҠйӮ»иҝ‘гҖӮ

3.5.2 Cacheзҡ„еҹәжң¬е·ҘдҪңеҺҹзҗҶ

  • еқ—й•ҝпјҡCacheеқ—пјҲиЎҢпјүдёҺдё»еӯҳеқ—еӨ§е°ҸзӣёзӯүгҖӮ
  • **е‘ҪдёӯзҺҮ(p)**пјҡ$p = \frac{\text{е‘Ҫдёӯж¬Ўж•°}}{\text{жҖ»и®ҝй—®ж¬Ўж•°}}$гҖӮ
  • е№іеқҮи®ҝй—®ж—¶й—ҙпјҡ$T_a = T_c + (1-p) \times T_m$пјҲ$T_c$пјҡCacheи®ҝй—®ж—¶й—ҙпјҢ$T_m$пјҡдё»еӯҳи®ҝй—®ж—¶й—ҙпјүгҖӮ

3.5.3 Cacheе’Ңдё»еӯҳзҡ„жҳ е°„ж–№ејҸ

1. зӣҙжҺҘжҳ е°„

  • жҳ е°„е…ізі»пјҡCacheиЎҢеҸ· = дё»еӯҳеқ—еҸ· mod CacheиЎҢж•°гҖӮ
  • дё»еӯҳең°еқҖж јејҸпјҡ| ж Үи®°(t) | иЎҢеҸ·(c) | еқ—еҶ…ең°еқҖ |
  • зү№зӮ№пјҡз®ҖеҚ•пјҢдҪҶеҶІзӘҒзҺҮй«ҳпјҢе‘ҪдёӯзҺҮдҪҺгҖӮ

2. е…ЁзӣёиҒ”жҳ е°„

  • жҳ е°„е…ізі»пјҡдё»еӯҳеқ—еҸҜжҳ е°„еҲ°д»»ж„ҸCacheиЎҢгҖӮ
  • дё»еӯҳең°еқҖж јејҸпјҡ| ж Үи®° | еқ—еҶ…ең°еқҖ |
  • зү№зӮ№пјҡеҶІзӘҒзҺҮдҪҺпјҢе‘ҪдёӯзҺҮй«ҳпјҢдҪҶзЎ¬д»¶ејҖй”ҖеӨ§пјҲйңҖжҜ”иҫғеҷЁпјүгҖӮ

3. з»„зӣёиҒ”жҳ е°„

  • жҳ е°„е…ізі»пјҡз»„й—ҙзӣҙжҺҘжҳ е°„пјҢз»„еҶ…е…ЁзӣёиҒ”жҳ е°„гҖӮ
  • дё»еӯҳең°еқҖж јејҸпјҡ| ж Үи®°(t) | з»„еҸ·(k) | еқ—еҶ…ең°еқҖ |
  • зү№зӮ№пјҡжҠҳдёӯж–№жЎҲпјҢ$r$и·Ҝз»„зӣёиҒ”еҚіжҜҸз»„$r$иЎҢгҖӮ
жҳ е°„ж–№ејҸ е‘ҪдёӯзҺҮ зЎ¬д»¶ејҖй”Җ еҶІзӘҒзҺҮ
зӣҙжҺҘжҳ е°„ жңҖдҪҺ жңҖе°Ҹ жңҖй«ҳ
е…ЁзӣёиҒ”жҳ е°„ жңҖй«ҳ жңҖеӨ§ жңҖдҪҺ
з»„зӣёиҒ”жҳ е°„ дёӯзӯү дёӯзӯү дёӯзӯү

3.5.4 Cacheдёӯдё»еӯҳеқ—зҡ„жӣҝжҚўз®—жі•

з®—жі• зү№зӮ№
йҡҸжңәз®—жі• йҡҸжңәжӣҝжҚўпјҢж— еұҖйғЁжҖ§дҫқжҚ®
е…Ҳиҝӣе…ҲеҮә(FIFO) жӣҝжҚўжңҖж—©и°ғе…Ҙзҡ„иЎҢпјҢж— еұҖйғЁжҖ§дҫқжҚ®
жңҖиҝ‘жңҖе°‘дҪҝз”Ё(LRU) жӣҝжҚўжңҖд№…жңӘи®ҝй—®зҡ„иЎҢпјҢеҲ©з”ЁеұҖйғЁжҖ§еҺҹзҗҶпјҢе‘ҪдёӯзҺҮй«ҳ

LRUи®Ўж•°еҷЁи§„еҲҷпјҲ$2^k$и·Ҝпјүпјҡ

  • е‘Ҫдёӯпјҡиў«и®ҝй—®иЎҢи®Ўж•°еҷЁжё…йӣ¶пјҢжҜ”е…¶дҪҺзҡ„еҠ 1пјҢе…¶дҪҷдёҚеҸҳгҖӮ
  • жңӘе‘Ҫдёӯдё”жңүз©әй—ҙпјҡж–°иЎҢи®Ўж•°еҷЁ=0пјҢе…¶дҪҷеҠ 1гҖӮ
  • жңӘе‘Ҫдёӯдё”ж— з©әй—ҙпјҡж·ҳжұ°и®Ўж•°еҖјжңҖеӨ§иЎҢпјҢж–°иЎҢи®Ўж•°еҷЁ=0пјҢе…¶дҪҷеҠ 1гҖӮ

3.5.5 Cacheзҡ„дёҖиҮҙжҖ§й—®йўҳ

еҶҷе‘Ҫдёӯж—¶

зӯ–з•Ҙ еҒҡжі• дјҳзјәзӮ№
е…ЁеҶҷжі• еҗҢж—¶еҶҷе…ҘCacheе’Ңдё»еӯҳ е®һзҺ°з®ҖеҚ•пјҢдёҖиҮҙжҖ§еҘҪпјҢдҪҶи®ҝеӯҳж¬Ўж•°еӨҡ
еӣһеҶҷжі• еҸӘеҶҷCacheпјҢжӣҝжҚўж—¶еҶҷеӣһдё»еӯҳ и®ҝеӯҳж¬Ўж•°е°‘пјҢйңҖи®ҫи„ҸдҪҚпјҢеӯҳеңЁдёҚдёҖиҮҙйҡҗжӮЈ

еҶҷдёҚе‘Ҫдёӯж—¶

зӯ–з•Ҙ еҒҡжі•
еҶҷеҲҶй…Қжі• жӣҙж–°дё»еӯҳпјҢе°Ҷеқ—и°ғе…ҘCache
йқһеҶҷеҲҶй…Қжі• еҸӘжӣҙж–°дё»еӯҳпјҢдёҚи°ғе…ҘCache

е…ёеһӢз»„еҗҲпјҡе…ЁеҶҷжі• + йқһеҶҷеҲҶй…Қжі•пјӣеӣһеҶҷжі• + еҶҷеҲҶй…Қжі•гҖӮ

3.5.6 жҢҮд»ӨCacheе’Ңж•°жҚ®CacheеҲҶзҰ»

  • дјҳзӮ№пјҡйҒҝе…ҚеҸ–жҢҮд»ӨдёҺеҸ–ж•°жҚ®зҡ„иө„жәҗеҶІзӘҒпјҢеҲ©з”ЁдёҚеҗҢеұҖйғЁжҖ§дјҳеҢ–жҖ§иғҪгҖӮ

3.5.7 Cacheе®№йҮҸи®Ўз®—

  • Cacheе®№йҮҸ = ж•°жҚ®еҢәе®№йҮҸ + ж Үи®°еҢәе®№йҮҸ
  • ж•°жҚ®еҢәе®№йҮҸ = еқ—еӨ§е°Ҹ Г— еқ—ж•°
  • ж Үи®°еҢәе®№йҮҸ = (ж Үи®° + жңүж•ҲдҪҚ + и„ҸдҪҚ + жӣҝжҚўжҺ§еҲ¶дҪҚ) Г— еқ—ж•°
  • ж Үи®°дҪҚдҪҚж•°пјҡ
    • зӣҙжҺҘжҳ е°„пјҡ$n - k - p$пјҲ$n$пјҡең°еқҖдҪҚж•°пјҢ$k$пјҡиЎҢеҸ·дҪҚж•°пјҢ$p$пјҡеқ—еҶ…ең°еқҖдҪҚж•°пјү
    • е…ЁзӣёиҒ”жҳ е°„пјҡ$n - p$
    • з»„зӣёиҒ”жҳ е°„пјҡ$n - (k-r) - p$пјҲ$r$пјҡз»„еҶ…иЎҢж•°жҢҮж•°пјү

3.6 иҷҡжӢҹеӯҳеӮЁеҷЁ

3.6.1 еҹәжң¬жҰӮеҝө

  • зӣ®зҡ„пјҡе°Ҷдё»еӯҳе’Ңиҫ…еӯҳз»ҹдёҖзј–еқҖпјҢеҪўжҲҗеӨ§е®№йҮҸең°еқҖз©әй—ҙгҖӮ
  • йҖ»иҫ‘ең°еқҖпјҲиҷҡең°еқҖпјүпјҡз”ЁжҲ·зј–зЁӢдҪҝз”Ёзҡ„ең°еқҖгҖӮ
  • зү©зҗҶең°еқҖпјҲе®һең°еқҖпјүпјҡе®һйҷ…дё»еӯҳеҚ•е…ғең°еқҖгҖӮ
  • зү№зӮ№пјҡз”ЁжҲ·зңӢжқҘжӢҘжңүиҫ…еӯҳзҡ„е®№йҮҸе’Ңдё»еӯҳзҡ„йҖҹеәҰгҖӮ

3.6.2 йЎөејҸиҷҡжӢҹеӯҳеӮЁеҷЁ

йЎөиЎЁ

  • йЎөиЎЁйЎ№еӯ—ж®өпјҡ
    • иЈ…е…ҘдҪҚпјҲжңүж•ҲдҪҚпјүпјҡйЎөйқўжҳҜеҗҰеңЁдё»еӯҳгҖӮ
    • дҝ®ж”№дҪҚпјҲи„ҸдҪҚпјүпјҡйЎөйқўжҳҜеҗҰиў«дҝ®ж”№пјҲеӣһеҶҷз”ЁпјүгҖӮ
    • дҪҝз”ЁдҪҚпјҲжӣҝжҚўжҺ§еҲ¶дҪҚпјүпјҡз”ЁдәҺжӣҝжҚўз®—жі•гҖӮ
    • еӯҳж”ҫдҪҚзҪ®пјҡзү©зҗҶйЎөеҸ·пјҲеңЁдё»еӯҳж—¶пјүжҲ–зЈҒзӣҳең°еқҖпјҲдёҚеңЁдё»еӯҳж—¶пјүгҖӮ

ең°еқҖиҪ¬жҚў

  • иҷҡең°еқҖпјҡ| иҷҡйЎөеҸ· | йЎөеҶ…еҒҸз§» |
  • зү©зҗҶең°еқҖпјҡ| зү©зҗҶйЎөеҸ· | йЎөеҶ…еҒҸз§» |
  • иҪ¬жҚўиҝҮзЁӢпјҡйЎөиЎЁеҹәеқҖеҜ„еӯҳеҷЁ вҶ’ йЎөиЎЁ вҶ’ иҷҡйЎөеҸ·зҙўеј• вҶ’ йЎөиЎЁйЎ№ вҶ’ зү©зҗҶйЎөеҸ· + йЎөеҶ…еҒҸз§»

еҝ«иЎЁ(TLB)

  • е°Ҷжҙ»и·ғзҡ„йЎөиЎЁйЎ№еӨҚеҲ¶еҲ°й«ҳйҖҹзј“еӯҳпјҲSRAMпјүпјҢйҮҮз”Ёе…ЁзӣёиҒ”жҲ–з»„зӣёиҒ”гҖӮ
  • дҪңз”ЁпјҡеҮҸе°‘и®ҝеӯҳж¬Ўж•°пјҢеҠ еҝ«ең°еқҖиҪ¬жҚўгҖӮ

еӨҡзә§еӯҳеӮЁзі»з»ҹпјҲTLB + Cacheпјүзҡ„CPUи®ҝеӯҳиҝҮзЁӢ

TLBгҖҒPageгҖҒCacheдёүз§ҚзјәеӨұжғ…еҶөпјҡ

еәҸеҸ· TLB Page Cache иҜҙжҳҺ
1 е‘Ҫдёӯ е‘Ҫдёӯ е‘Ҫдёӯ дҝЎжҒҜеңЁCacheдёӯ
2 е‘Ҫдёӯ е‘Ҫдёӯ зјәеӨұ дҝЎжҒҜеңЁдё»еӯҳпјҢдёҚеңЁCache
3 зјәеӨұ е‘Ҫдёӯ е‘Ҫдёӯ дҝЎжҒҜеңЁдё»еӯҳе’ҢCacheдёӯ
4 зјәеӨұ е‘Ҫдёӯ зјәеӨұ дҝЎжҒҜеңЁдё»еӯҳпјҢйңҖи®ҝеӯҳдёӨж¬Ў
5 зјәеӨұ зјәеӨұ зјәеӨұ зјәйЎөпјҢйңҖи®ҝй—®зЈҒзӣҳ

3.6.3 ж®өејҸиҷҡжӢҹеӯҳеӮЁеҷЁ

  • ж®өпјҡжҢүзЁӢеәҸйҖ»иҫ‘з»“жһ„еҲ’еҲҶпјҢй•ҝеәҰеҸҜеҸҳгҖӮ
  • иҷҡең°еқҖпјҡ| ж®өеҸ· | ж®өеҶ…ең°еқҖ |
  • ж®өиЎЁпјҡи®°еҪ•ж®өиө·зӮ№гҖҒж®өй•ҝгҖҒиЈ…е…ҘдҪҚзӯүгҖӮ
  • дјҳзӮ№пјҡйҖ»иҫ‘зӢ¬з«ӢпјҢжҳ“дәҺз®ЎзҗҶгҖҒдҝқжҠӨгҖӮ
  • зјәзӮ№пјҡж®өй—ҙзўҺзүҮпјҢз©әй—ҙжөӘиҙ№гҖӮ

3.6.4 ж®өйЎөејҸиҷҡжӢҹеӯҳеӮЁеҷЁ

  • е…ҲжҢүйҖ»иҫ‘з»“жһ„еҲҶж®өпјҢжҜҸж®өеҶҚеҲҶйЎөгҖӮ
  • иҷҡең°еқҖпјҡ| ж®өеҸ· | ж®өеҶ…йЎөеҸ· | йЎөеҶ…ең°еқҖ |
  • иҪ¬жҚўиҝҮзЁӢпјҡж®өиЎЁ вҶ’ йЎөиЎЁ вҶ’ зү©зҗҶйЎөеҸ· + йЎөеҶ…ең°еқҖгҖӮ

3.6.5 иҷҡжӢҹеӯҳеӮЁеҷЁдёҺCacheзҡ„жҜ”иҫғ

еҜ№жҜ”йЎ№ Cache иҷҡжӢҹеӯҳеӮЁеҷЁ
дё»иҰҒзӣ®зҡ„ и§ЈеҶійҖҹеәҰй—®йўҳ и§ЈеҶіе®№йҮҸй—®йўҳ
е®һзҺ°ж–№ејҸ зЎ¬д»¶ зЎ¬д»¶ + OS
йҖҸжҳҺжҖ§ еҜ№жүҖжңүзЁӢеәҸе‘ҳйҖҸжҳҺ еҜ№еә”з”ЁзЁӢеәҸе‘ҳйҖҸжҳҺпјҢеҜ№зі»з»ҹзЁӢеәҸе‘ҳдёҚйҖҸжҳҺ
зјәеӨұд»Јд»· и®ҝй—®дё»еӯҳ и®ҝй—®зЈҒзӣҳпјҲд»Јд»·жӣҙеӨ§пјү
ж•°жҚ®жөҒ дё»еӯҳ вҶ’ Cache вҶ’ CPU зЈҒзӣҳ вҶ’ дё»еӯҳ вҶ’ CPU

3.6.6 ж•°жҚ®зҡ„еҜ№йҪҗеӯҳеӮЁ

иҫ№з•ҢеҜ№йҪҗ

  • иҰҒжұӮеӯҳеӮЁең°еқҖжҳҜж•°жҚ®иҮӘиә«еӨ§е°Ҹзҡ„ж•ҙж•°еҖҚгҖӮ
  • дјҳзӮ№пјҡдёҖж¬Ўи®ҝеӯҳеҚіеҸҜеҸ–еҮәж•°жҚ®пјҲз©әй—ҙжҚўж—¶й—ҙпјүгҖӮ

з»“жһ„дҪ“зҡ„иҫ№з•ҢеҜ№йҪҗ

  • еҜ№йҪҗи§„еҲҷпјҡ
    • жҜҸдёӘжҲҗе‘ҳжҢүиҮӘиә«зұ»еһӢеӨ§е°ҸеҜ№йҪҗпјҲchar:1, short:2, int:4пјүгҖӮ
    • з»“жһ„дҪ“й•ҝеәҰдёәжңҖеӨ§жҲҗе‘ҳеҜ№йҪҗеҖјзҡ„ж•ҙж•°еҖҚгҖӮ
